sea.js中加载layer,缺少css文件解决

发表于 2017-02-08 | 分类于 遇到的问题 | 阅读次数

今天做项目时遇到了一个问题,看下图

页面错乱,询问前端工程师后,定位问题应该是缺少layer.css导致的,这就很奇怪了,之前的项目使用layer的时候,也只是单单的引入jquery和layer.min.js
为什么现在这个新项目就不行了呢？

手动配置layer的路径,可以是绝对路径也可以是相对路径,只要能找得到layer的文件夹
